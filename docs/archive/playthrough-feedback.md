## Playthrough feedback

1. The aim of the shot seems to snap to some increment.  I would like this to be dramatically increased in precision.  I would like this to be a real skill thing to be able to get the bubble to just squeak into a crack between landed bubbles.  

2. The landing of the bubbles snaps to a grid.  This should land more naturally allowing for drift between structures leaving irregular sized spaces that enable fun gameplay mechanics.

3. There is something to the ease of the current game, it is rather nice to have the bubble snap into the perfect spot easily.  Perhaps we can save this as a "Special mechanic" power up called "Sure Shot"

4. Lets add some level progressions, scores and stats (using arcade framework). 

5. The game is still a bit to challenging for a beginner level.  This level of progression is challenging to fend off at the current descent rate.  

6. I wonder if descent being time related would be a better mechanic, rewarding quick decisions and placment

7. Lets add other power ups that can help save a player if used wisely

8. I would like to replace the current cannon/gun ui element quickly, it is getting annoying having this overlap the loaded bubble.

9. Lets get a prototype of the boatman going.  I want to see what this animation could look like.
